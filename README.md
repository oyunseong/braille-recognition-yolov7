### 🧠 실환경을 고려한 경량화된 점자인식 모델

실환경 촬영 조건을 고려한 점자 인식 딥러닝 모델 연구

시각장애인의 점자 해독 보조를 목표로 진행한 정보통신공학과 졸업 논문 프로젝트

---

### 💡 Topic

“실환경을 고려한 경량화된 점자인식 모델” 이라는 주제로
정보통신공학과 졸업 논문을 작성하였습니다.

본 연구는 시각 장애인들이 실제 환경에서 점자를 인식하고 해독하는 데 겪는 어려움을 해결하기 위해,

•	실내·실외 환경

•	촬영 각도 및 거리

•	조도(밝기) 변화

등 실환경 요소를 고려한 점자 인식 모델을 제안하는 것을 목표로 진행되었습니다.

--- 

### 🗓️ 연구 기간

2022.08 ~ 2022.12

--- 

### 📝 Summary

•	본 논문에서는 YOLOv7-w6 모델을 기반으로 점자 객체 검출 모델을 설계하여 기존 연구 대비 높은 정확도와 빠른 추론 속도를 달성하였습니다.

•	실제 사용 환경을 고려하여 촬영 각도 / 거리 / 밝기 조건별 점자 데이터셋을 직접 구축하였습니다.

•	각 조건에서의 모델 성능을 비교 분석하여 사람이 실제로 점자를 인식할 수 있는 거리 및 각도 범위를 수치적으로 분석하였습니다.

•	최종적으로는 해당 모델을 활용한 모바일 어플리케이션 프로토타입을 구현하여 시각 장애인의 점자 해독 보조 가능성을 검증하였습니다.

--- 

### ✨ 주요 기능

🔹 YOLOv7-w6 기반 점자 추론

•	객체 탐지(Object Detection) 방식으로 점자 위치 인식

•	기존 CNN 기반 분류 방식 대비 높은 실환경 대응력 확보

--- 

🔹 모바일 앱 연동 추론 구조

1.	Android 앱에서 점자 이미지 촬영
2.	촬영 이미지를 Google Colab Server로 전송
3.	서버에서 YOLOv7 모델 추론 수행
4.	추론 결과(JSON) 반환
5.	앱 UI에 점자 결과 표시

--- 

🔹 음성 출력 (TTS)

•	추론된 점자 문자를 Google Text-to-Speech(TTS) 를 통해 음성으로 출력

•	시각장애인의 실사용 환경을 고려한 기능 구현
